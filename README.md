## Quick Start

### Using docker (recommended)
Build docker image

```bash
docker build -t terminal-resume .
```

Run container

```bash
docker run -p 3000:3000 terminal-resume
```

### Using npm/yarn

1. Install dependencies:

```bash
yarn install
```

2. Build the project:

```bash
yarn build
```

3. Run the server:

```bash
yarn start
```
